## Instalation

1 - Clone repository using ssh key
    
```bash
$ git clone git@github.com:joaohenriquepda/credi-discount.git
```
2 - Change to the project directory:

```bash
$ cd credi-discount
```
3 - Use docker-compose to start local development environment

```bash
$ docker-compose up
```
